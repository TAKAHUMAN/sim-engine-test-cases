# Notebooks

Notebook placeholders are reserved for interactive analyses:

1. `01_pk_reproduction.ipynb`: Mizuno PK reproduction and pcVPC checks.
2. `02_pd_development.ipynb`: Biomarker and tumor-growth development.
3. `03_calibration_results.ipynb`: Resistance calibration and trace review.
4. `04_publication_figures.ipynb`: Polished figures.

The production implementation lives in `model/` and `simulation/`.
